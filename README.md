# SpineSight XR

### Overview
SpineSight XR is a HoloLens 2 application designed for school-based scoliosis screening. It digitizes the traditional Adam’s Forward Bend Test by mapping the spinal column in 3D space to identify curvature before it requires invasive surgery.

### Core Functionality
* **Spatial Mapping:** Leverages HoloLens hand-tracking to place holographic fiduciaries along the vertebrae.
* **Biometric Calculation:** Generates a 3D spline (center line) to calculate lateral deviation and surface Cobb angles.

### Technical Stack
* **Hardware:** Microsoft HoloLens 2
* **Engine:** Unity + MRTK (Mixed Reality Toolkit)
* **Language:** C#

### Value Proposition
Standard school screenings are subjective and prone to human error. SpineSight XR provides objective, repeatable data to help school nurses identify at-risk students and accelerate referrals to orthopedic specialists.
